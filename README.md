# crealib-accelerator

Fork de Seraphinite Accelerator.

## Scripts:

### actualizar-accelerator
Haciendo uso de la licencia GPL, este modifica el código original de la versión gratuita de Seraphinite Accelerator para eliminar el banner publicitario del footer. 

### actualizar-accelerator-cron
Automatiza el proceso de actualización para todos los WordPress instalados en el servidor que tengan instalado Seraphinite.
